Para inicializar:
inicializar un servidor y al menos un cliente, poner IP del server

ingresar nick de usuario para ingresar a la aplicacion.

Comandos:

MSG [destino] [mensaje]  para enviar mensaje a otro usuario conectado
CRE [usuario] crear: agregar a un usuario como contacto
CTS [usuario] ver: muestra los contactos de ese usuario

Los mensajes y contactos se guardan en carpetas, todo esta en la carpeta "data"
